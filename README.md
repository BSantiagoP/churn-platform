# Customer Intelligence & Churn Prediction Platform

## Objective
Build an end-to-end, production-style ML platform that ingests raw customer data,
transforms it into analytics-ready features, trains and evaluates churn models,
and produces actionable insights for business stakeholders.

## Tech Stack
- Databricks (Free Edition)
- Delta Lake (Bronze / Silver / Gold)
- Python (PySpark, MLflow, sklearn)
- Databricks SQL
- GitHub

## Project Structure
This repository follows a production-oriented structure separating ingestion,
transformation, modeling, and inference layers.
