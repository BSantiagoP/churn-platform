## Gold Features

### Numeric
- tenure: customer tenure in months
- MonthlyCharges: monthly billing amount
- TotalCharges: cumulative charges
- SeniorCitizen: binary indicator

### Binary Encoded
- Partner, Dependents, PhoneService, PaperlessBilling (0/1)

### One-Hot Encoded
- gender
- MultipleLines
- InternetService
- OnlineSecurity
- OnlineBackup
- DeviceProtection
- TechSupport
- StreamingTV
- StreamingMovies
- Contract
- PaymentMethod

### Label
- label_churn: 1 = churned, 0 = retained
